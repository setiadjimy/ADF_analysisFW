# ADF_analysisFW
This repository contains the source code for the automated forensic triage and causal verification framework. This tool provides a structured approach to the analysis phase of automotive digital forensics.

Because I do not have a laptop with a high-powered graphics card to run CARLA, a trade-off must be made. For this experiment, CARLA 0.9.12 is used because the Unreal Engine in that version can run on a graphics card with 4GB of memory. Please consider this if you want to replicate the experiment.
